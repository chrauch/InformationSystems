# Information Systems / Data Management

This repository provides an overview of Information Systems and Data Management topics. It contains supplementary teaching material for students and personal notes. The correctness of the material provided cannot be guaranteed. Feedback is welcome, and errata can be sent to the author.

## Sources

Parts of this material are based on the *Information Systems / Database Systems* lectures by Jun-Prof. Dr. Panagiotis Bouros (Johannes Gutenberg University Mainz). Additional material originates from the accompanying lab course taught by the author.
